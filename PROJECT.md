# Pokémon Clone - Project Plan

## Overview
A GameBoy-style Pokémon clone built with modern web tech, deployed to Vercel.

## Tech Stack
- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- HTML5 Canvas for game rendering
- Vercel for deployment

## Domains
- pokemon.nak.im
- pokemon.nei.ma

## Core Features (MVP)
1. **Overworld** - Tile-based movement, collision detection
2. **Battle System** - Turn-based combat with types
3. **Pokémon** - Catch, train, evolve
4. **Inventory** - Items, Pokéballs, potions
5. **Save System** - Local storage + cloud sync

## Visual Style
- GameBoy Color aesthetic (8-bit, limited palette)
- 160x144 resolution scaled up
- Authentic UI patterns
